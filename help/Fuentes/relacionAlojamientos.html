Relacion Alojamientos

La pantalla permite al cliente seleccionar alojamientos asociados al parque tematico que ha escogido.

Una vez escogido uno el cliente tendra que pasar a introducir los datos para indicar como quiere el alojamiento.

Datos a introducir:

  1. Nº Adultos: el cliente debera de indicar el numero de adultos para los que quiere reservar el paquete.
  2. Nº Niños: el cliente debera de indicar el numero de niños para los que quiere reservar el paquete.
  3. Fecha Reserva: el cliente tendra que indicar para cuando quiere la reserva.
  4. Nº dias: el cliente tendra que indicar el numero de dias que quiere reservar el alojamiento.
  5. Desayuno incluido: en el caso de que el alojamiento escogido sea un hotel se podra indicar si se quiere o no tener el desayuno incluido, esto incrementea el precio del alojamiento en un 10% por persona y dia.

Tras introducir los datos el cliente podra añadir el alojamiento a la reserva e indicar las observaciones que estime oportunas.

Si no quiere reservar nada mas daria a siguiente y pasaria a formalizar la reserva.
