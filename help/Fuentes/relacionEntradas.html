Relacion Entradas

La pantalla permite al cliente seleccionar entradas asociados al parque tematico que ha escogido.

Una vez escogida una el cliente tendra que pasar a introducir los datos para indicar como quiere la entrada.

Datos a introducir:

  1. Nº Adultos: el cliente debera de indicar el numero de adultos para los que quiere reservar el paquete.
  2. Nº Niños: el cliente debera de indicar el numero de niños para los que quiere reservar el paquete.
  3. Fecha Reserva: el cliente tendra que indicar para cuando quiere la reserva.
  4. Nº dias: el cliente tendra que indicar el numero de dias que quiere reservar el alojamiento.

Tras introducir los datos el cliente podra añadir la entrada a la reserva e indicar las observaciones que estime oportunas.

Si no quiere reservar nada mas daria a siguiente y pasaria a formalizar la reserva.
