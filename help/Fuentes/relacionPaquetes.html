Relacion paquetes

La pantalla permite al cliente seleccionar paquetes asociados al parque tematico que ha escogido.

Una vez escogido uno el cliente tendra que pasar a introducir los datos para indicar como quiere el paquete.

Datos a introducir:

  1. Nº Adultos: el cliente debera de indicar el numero de adultos para los que quiere reservar el paquete.
  2. Nº Niños: el cliente debera de indicar el numero de niños para los que quiere reservar el paquete.
  3. Fecha Reserva: el cliente tendra que indicar para cuando quiere la reserva.

Tras introducir los datos el cliente podra añadir el paquete a la reserva e indicar las observaciones que estime oportunas.

Si no quiere reservar nada mas daria a siguiente y pasaria a formalizar la reserva.
