Reserva

Una vez escogido todo lo que el cliente quiera, se pasa a formalizar para ello debera introducir:

  1. Nombre: el cliente debera de introducir a nombre de quien se quiere hacer la reserva.
  2. Apellidos: el cliente debera de indicar los apellidos de para quien es la reserva.
  3. DNI: el cliente debera de indicar el DNI de para quien es la reserva.

Una vez hecho esto se dara a generar recibo, se visualizara en pantalla el resumen de la reserva y se podra pasar a guardar la reserva en el caso de que todo este correcto.
